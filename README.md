# Virtual Library

Static project scaffold for a single-origin library of classic works rendered with PreTeXt and wrapped by a vanilla TypeScript bookshelf shell.

## Build

```sh
uv run pretext --version
npm install
npm run build
```

`make build` runs the Vite shell build into `output/`. Scraping, catalog normalization, reader features, Pagefind indexing, portraits, and end-to-end tests land in later briefs.
